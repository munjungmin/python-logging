# Python Logging 학습 저장소

Python 로깅을 단계별로 학습한 내용을 브랜치별로 정리한 저장소입니다.

## 브랜치 소개

| 브랜치 | 내용 |
|---|---|
| `creating_child_loggers` | 핸들러를 이용한 로그 출력 위치 지정 및 child logger 생성 |
| `handling_exceptions` | 예외 발생 시 로그 출력 처리 |
| `structured_logging` | structlog을 이용한 JSON 형식의 구조화 로깅 |
| `context_binding` | structlog contextvars로 요청별 컨텍스트 자동 바인딩 |
| `logging_middleware` | FastAPI 미들웨어로 로깅 관련 코드를 공통으로 처리 |
